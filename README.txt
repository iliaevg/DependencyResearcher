# DependencyResearcher

Реализован функцонал, указанный задании:

Разработать программу на языке Java для обработки файла, в котором задаются
зависимости сущностей друг от друга. Каждая сущность представлена своим
уникальным идентификатором в виде целого числа. Зависимости задаются
строкой в текстовом файле, содержащем произвольное количество строк формата:
<id сущности> <id сущности от которой она зависит>. Файл читать из STDIN. Для
прочитанных сведенй о связях между сущностями необходимо найти циклические
зависимости и вывести их в STDOUT в виде id сущностей через пробел.
Пример файла​:
1 2
2 1
3 4
5 6
6 5

Ожидаемый вывод в STDOUT:​
1 2 1
5 6 5


Проект выложен на github по ссылке:
https://github.com/iliaevg/DependencyResearcher/

По сборке и запуску.
Задание реализовано с помощью IDE NetBeans 8.0.2 на платформе jdk1.7.0_80. 
Каталог DependencyResearcher - кореновой для проекта
Для сборки выполнить команду
    ant jar
для запуска програмы с определенным входным файлом использовать команду
    java -jar dist\DependencyResearcher.jar "имя_файла"
для запуска програмы со входным с клавиатуры использовать команду
    java -jar dist\DependencyResearcher.jar

так же можно запускать проект командой
    ant run 
В этом случае программа будет считывать данные с файла dependencies.txt в корне
проекта
